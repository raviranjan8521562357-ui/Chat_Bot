# 💬 LangChain Chatbot using Gemini API

An interactive AI chatbot built with **LangChain**, **Google Gemini API**, and **Streamlit**.  
It supports real-time conversations, maintains chat context using Streamlit session state,  
and demonstrates integration of large language models with a web UI.
